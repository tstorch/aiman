# Goose (OpenAI-kompatibel) – Verwendung

- Setze die Basis-URL über `OPENAI_API_BASE` (z. B. GooseAI Endpunkt) und nutze einen OpenAI-kompatiblen Client.
- Verwende den gerenderten Prompt als Eingabe; beachte Ratenlimits und Token-Grenzen.

Agent-spezifische Hinweise:

- Prüfe Kompatibilität der Chat-/Completion-Formate; ggf. System-Prompt separat übergeben.
