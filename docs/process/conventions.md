# Projektkonventionen

## Benennung & IDs

- IDs: `YYYYMMDD-HHMMSS-rand` (automatisch via Skript)
- Slugs: `kebab-case`

## Statuswerte

- Gemäß `config/project.yml` (draft, proposed, in-progress, blocked, review, done, deprecated)

## Verlinkung

- Eltern-Kind-Beziehungen via `parent: <ID>`
- Relative Pfade zwischen Artefakten benutzen
